dotfiles and toy tools.

REQUIREMENTS
============

* rake
* python
* ruby

at ubuntu desktop(python package is already installed:):

  sudo apt-get install rake

SETUP
=====

Ubuntu::

  wget "https://github.com/meiraka/local/blob/master/dep/ubuntu.sh"
  chmod +x ubuntu.sh
  ./ubuntu.sh
  chsh -s /bin/zsh
